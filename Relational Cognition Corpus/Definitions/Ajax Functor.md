# Adjoint-Lax (Ajax) Functor

Type: definition
Tags: node/definition, domain/cat, cat/relations, logic/regular, status/active

Definition
- A lax symmetric monoidal po-functor F: C → D whose laxators are right adjoints. Ajax functors preserve adjoint monoids.
