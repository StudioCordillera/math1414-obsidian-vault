# Relations Po-Category (Rel_R)

Type: definition
Tags: node/definition, domain/cat, cat/relations, logic/regular, status/active

Definition
- For a regular category R, Rel_R has objects Ob(R) and morphisms x: r ⇸ s given by subobjects x ⊆ r×s. Composition is by pullback + image factorization; monoidal structure inherited from cartesian product.
