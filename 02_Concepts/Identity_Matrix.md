# Identity Matrix

Definition
- The identity matrix I_n is the n×n matrix with ones on the main diagonal and zeros elsewhere. It satisfies I_n A = A I_n = A for any n×n matrix A.

Use
- Acts as multiplicative identity; central to defining inverses.
