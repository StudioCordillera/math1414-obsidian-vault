# Common Logarithm (base 10)

Definition
- The common logarithm is log(x) = log_{10}(x), the inverse of 10^x. It satisfies 10^{log x} = x for x>0 and log(10^x) = x.

Core facts
- Domain: (0, ∞); Range: (−∞, ∞)
- Identities: log(1) = 0, log(10) = 1
- Change of base: log_b(x) = log x / log b

Uses
- Scientific notation and orders of magnitude; calculators typically provide a log key.

Common pitfalls
- Using log for other bases without change-of-base; attempting log of nonpositive inputs.
