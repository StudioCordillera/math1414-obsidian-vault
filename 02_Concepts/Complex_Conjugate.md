# Definition
For z = a + bi, the complex conjugate is z̄ = a − bi.

# Properties
- z z̄ = a^2 + b^2 (real, nonnegative)
- (z̄)̄ = z; (z+w)̄ = z̄ + w̄; (zw)̄ = z̄ w̄
