# Matrix Multiplication

Definition/Method
- For A (m×n) and B (n×p), the product AB is m×p with entries (AB)_{ij} = Σ_k A_{ik} B_{kj}.

Properties
- Associative (A(BC)=(AB)C); distributive over addition; not commutative in general.
