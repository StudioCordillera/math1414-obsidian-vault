### Node Types (from Node Standard)
- **Topic**: Aggregate content about a subject
- **Definition**: Atomic concept (one per note)
- **Claim**: Theorem, property, or rule
- **Example**: Worked problem with solution
- **Method**: Step-by-step procedure
- **Question**: Problem or exercise

### Universal Knowledge Framework Approach
1. **Explicit Knowledge**: Definitions, formulas, procedures (stated directly)
2. **Implicit Knowledge**: Prerequisites, assumptions, unstated steps
3. **Common Errors**: What NOT to do
